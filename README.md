# A* Heuristics for the Traveling Salesman Problem

This project investigates the use of **A* search** for solving the **Traveling Salesman Problem (TSP)** and focuses on designing and evaluating improved heuristic functions to enhance search efficiency.

The implementation compares multiple heuristics to analyze their impact on runtime, node expansions, and solution quality. The project also explores the possibility of using a **machine learning model to estimate heuristic values** as an optional extension.

---

## Project Objectives

* Implement A* search for the Traveling Salesman Problem
* Compare different heuristic strategies
* Evaluate performance based on runtime and search efficiency
* Analyze strengths and limitations of heuristic approaches
